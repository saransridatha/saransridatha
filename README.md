* **Scholr 1.0**: Led infrastructure deployment for a multi-tenant ERP platform serving 1,200+ users. Orchestrated rootless containers on Linux VPS and built automated deployment workflows. `Podman` `Jenkins`
* **[SkySafe](https://github.com/saransridatha/SkySafe)**: Aviation Risk Intelligence Platform synthesizing disparate data vectors into real-time safety insights with interactive 3D mapping. `Next.js` `Three.js` `Gemini 2.0`
